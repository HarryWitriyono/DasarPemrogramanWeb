Beranda

Selamat datang di beranda web site ini. Website ini merupakan contoh penggunaan frameset, frame dan berbagai tag html yang sering anda gunakan pada pembangunan suatu aplikasi web. Beberapa tag HTML tersebut antara lain adalah :

  • Tag <img src="path/filefoto.jpg" width="lebar" height="panjang"> digunakan untuk menampilkan foto atau gambar pada halaman web.
  • Tag <audio controls autoplay> dan </audio> digunakan untuk menambahkan komponen pemutar suara pada halaman web. Tag ini diikuti dengan tag <source src="suara/suara.mp3"> sebagai penentu file suaranya.
  • Tag <video controls autoplay> dan </video> digunakan untuk menambahkan komponen pemutar video pada halaman web. Tag ini diikuti dengan tag <source src="video/video.mp4"> sebagai penentu file videonya.
  • Tag <ul> yang digunakan untuk membuat daftar tanpa penomoran, kebalikannya adalah tag <ol> bila kita ingin daftarnya mempunyai nomor urut.
  • Tag <li> digunakan untuk menentukan urutan pada jenis tag daftar yang digunakan.
  • Tag <a href="url link" target="nama target">Caption<a> digunakan untuk membuat penghubung ke halaman web lain, baik internal ataupun eksternal.
